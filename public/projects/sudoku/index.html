Grandma's Sudoku Robot

The Problem

My grandma loved doing Sudoku puzzles online, but her favorite website started running intrusive ads. As she got older, navigating these ads and the website became frustrating and difficult.

The Solution

I built her a simple desktop application - a .exe file on her desktop that generates personalized Sudoku puzzles as PDFs and automatically sends them to her printer. Each puzzle includes:

  • Messages from family members
  • Inspirational quotes

It's a simple PDF generator with a printer runner. Now she just clicks the icon, and a fresh puzzle prints out automatically. Simple, personal, and made with love.

Technical Implementation

Python GUI application using tkinter for interface design. Implements backtracking algorithm for Sudoku generation with configurable difficulty parameters. ReportLab handles PDF rendering with custom formatting for family messages and quote injection. System print queue integration enables automatic output to default printer. Packaged as standalone Windows executable via PyInstaller with embedded Python runtime.
